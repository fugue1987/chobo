## 约定与注意

- **`provider` = 计费渠道,不是厂商**:豆包 Ark 直连记 `doubao`;经网关中转记 `newapi`。
- **别名归一**:带日期/接入点后缀的 `request_model`(如 `doubao-seed-2-1-pro-260628`)经 `model_aliases` 归一到 canonical 再精确匹配 `(provider, model, operation)`。
- **币种原样存、看板分币种展示、永不跨币种相加**(¥ 与 $ 各算各的)。
- **缺价 = 诚实留空**:价目表没有对应三元组 → `total_cost=NULL` + 告警,绝不填 0。
- **按小时的缓存"存储"费不建模**:doubao 的 `缓存存储 0.017 元/MTok/时` 是按**时间**计的存储费;chobo 按 token 计、不按时,故不入表(表里的"缓存命中"是按-token 的读价)。
- **doubao-seed-2.1-pro 目前无输入分档**(单一价);若日后 Ark 出分档,再补分档行。
- **推理 token**:gemini-3.5-flash 单列 `reasoning` 费率(9.0);其余模型的推理并入输出计。
- gpt-image-2 逐模态 token 计价:文本输入按 `5.0`、图像输入按 `8.0`、图像输出按 `30.0`(USD)。

> **加新模型价**:改 `price-seed.json`(bump `version` + 追加价目行 / 别名)→ 常规部署走 `deploy/ship-crm.sh`(boot 自动 `syncPriceSeed` 引入新版本);运行中免重启热更新用 `seed-cli` / 裸 Node `update-prices.sh`(≤ `CHOBO_PRICE_REFRESH_SEC` 秒自动生效)。字段含义见 `deploy/customer/bare-node/交付指南.md`。
